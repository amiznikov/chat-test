# Socket.io Chat

## Start up application

This app is based on open source solution
```js
git clone https://github.com/harryheman/Socket.io-React-Chat-App.git
```

- Install npm/yarn packages

```bash
yarn
# or
npm i

cd client

yarn
# or
npm i

cd ..
cd server

yarn
# or
npm i
```

- For starting app

```bash
yarn start
# or
npm start
```
